#Расчет Пластинчатого теплообменника

##Тепловой расчет подогревателя

Параметры теплоносителей находят по табл. 1 и 7 (прил. 2) при средних температурах масла и пара

{
	t_мср=0.5*(t_м_l+t_м_ll) % \celsius
}

{
	t_пср=0.5*(t_п+t_н) % \celsius
}

Количество теплоты, передаваемой в единицу времени

{
	Q=D_п*C_п*(t_п-t_н)+D_п*r % кВт
}

Расход нагреваемого масла заданным количеством пара составит

{
	G_м=(Q)/(c_м*(t_м_ll-t_м_l)) % кг/с
}

Объемный расход масла

{
	V_м=G_м/ro_м % м**3/с
}

Температурные напоры

{
	dt_б=t_п-t_м_l % \celsius
}

{
	dt_м=t_н-t_м_ll % \celsius
}

Средний логарифмический температурный напор между теплоносителями
рассчитывают по формуле

{
	dt_ср=(dt_б-dt_м)/(ln(dt_б/dt_м)) % \celsius
}

Расчет поверхности теплообмена ведем методом последовательных при-
ближений

Первое приближение. Для определения ориентировочной площади поверхности 
теплообмена аппарата задаются значением коэффициента теплопередачи для двух 
теплоносителей — конденсирующийся пар, масло {k_ор=}

Ориентировочная поверхность теплообмена

{
	F_ор=(Q*10**3)/(k_ор*dt_ср) % м**2
}

По табл. 9 (прил. 2) принимаем пластинчатый теплообменник с разборными
сдвоенными пластинами (прил. 2, табл. 10), с площадью теплообмена {F_ор=}

Удельная тепловая нагрузка аппарата приблизительно будет

{
	q=(Q*10**3)/F_ор % Вт/м**2
}

В зависимости от удельной тепловой нагрузки критерий Рейнольдса для
стекающей пленки конденсата определяют по формуле

{
	Re_к=(q*L_п)/(r*10**3*ro_к*nu_к)
}

Критерий Нуссельта при условии конденсации пара для пластины ПР-0,5Е
(прил. 2, табл. 12) определяют по формуле

{
	Nu_к=240*(Re_к**0.7)*(Pr_к**0.4)
}

Коэффициент теплоотдачи от пленки конденсата к стенке

{
	al_к=(Nu_к*la_к)/L_п % Вт/(м**2*\celsius)
}

Рассчитывают среднюю температуру стенки

{
	t_сср=(t_мср+t_пср)/2 % \celsius
}

Скорость движения теплоносителя

{
	om_м=2*((al_м*(t_сср-t_мср)*dp)/(c_м*10**3*(t_м_ll-t_м_l)*ro_м**2*Ks_м))**(1/3) % м/с
}

Критерий Рейнольдса для потока масла

{
	Re_м=(om_м*d_э)/nu_м
}

Проверяют принятый коэффициент общего гидравлического сопротивления

{
	Ks_м=22.4*Re_м**(-0.25)
}

Поскольку значение коэффициента гидравлического сопротивления 
отличается от принятого более чем на 5 процентов, то выполняют поправочный пере-
расчет скорости.

Принимаем {Ks_м=} ; {al_м=}

Скорость движения теплоносителя

{
	om_м=2*((al_м*(t_сср-t_мср)*dp)/(c_м*10**3*(t_м_ll-t_м_l)*(ro_м**2)*Ks_м))**(1/3) % м/с
}

Критерий Рейнольдса для потока масла

{
	Re_м=(om_м*d_э)/nu_м
}

Коэффициент общего гидравлического сопротивления

{
	Ks_м=22.4/Re_м**0.25
}

Полученный коэффициент общего гидравлического сопротивления удов-
летворяет заданному значению.

Критерий Нуссельта для потока масла (см. прил. 2, табл. 12)

{
	Nu_м=0.13*Re_м**0.73*Pr_м**0.43*(Pr_м/Pr_ст)**0.25
}

Коэффициент теплоотдачи от стенки к нагреваемому маслу

{
	al_м=(Nu_м*la_м)/d_э % Вт/(м**2*\celsius)
}

Коэффициент теплопередачи в аппарате

{
	k=1/(1/al_к+de_ст/la_ст+1/al_м) % Вт/(м**2*\celsius)
}

Площадь поверхности теплопередачи в первом приближении

{
	F=(Q*10**3)/(k*dt_ср) % м**2
}

Так как полученная площадь поверхности теплообмена отличается от заданной
площади то принимают новое значение поверхности теплообмена и делают перерасчет

Принимаем {F_ор=}

Величина удельной тепловой нагрузки

{
	q=(Q*10**3)/F_ор % Вт/м**2
}

Число Рейнольдса для пленки конденсата

{
	Re_к=(q*L_п)/(r*10**3*ro_к*nu_к)
}

Критерий Нуссельта для конденсата

{
	Nu_к=240*(Re_к**0.7)*(Pr_к**0.4)
}

Коэффициент теплоотдачи от пленки конденсата к стенке

{
	al_к=(Nu_к*la_к)/L_п % Вт/(м**2*\celsius)
}

Коэффициент теплопередачи в аппарате

{
	k=1/(1/al_к+de_с/la_с+1/al_м) % Вт/(м**2*\celsius)
}

Площадь поверхности теплопередачи во втором приближении

{
	F=(Q*10**3)/(k*dt_ср) % м**2
}

##Компоновочный расчет и уточнение рабочей поверхности теплообменного аппарата

Площади поперечного сечения пакетов теплообменника со стороны масла
находят по формуле

{
	f_пм=V_м/om_м % м**2
}

Число каналов в одном пакете для масла

{
	m_м=f_пм/f_1
}

Поверхность теплообмена одного пакета

{
	F_п=F_1*n_п % м**2
}

Число пакетов теплообменника со стороны масла

{
	X_м=F/F_п
}

Первый Вариант {X_м=}

Общее число пластин в аппарате

{
	n_п=(F+2*F_1)/F_1
}

тогда число каналов в аппарате

{
	m=n_п-1
}

Принимаем {m_м=}

При этом фактическая площадь поперечного сечения пакета со стороны
масла

{
	f_пм=m_м*f_1 % м**2
}

Фактическая скорость движения масла в каналах

{
	om_м=V_м/f_пм % м/с
}

При этих условиях определяем параметры

{
	Re_м=(om_м*d_э)/nu_м
}

{
	Nu_м=0.135*Re_м**0.73*Pr_м**0.43*(Pr_м/Pr_ст)**0.25
}

{
	al_м=(Nu_м*la_м)/d_э % Вт/(м**2*\celsius)
}

{
	k=1/(1/al_к+de_ст/la_ст+1/al_м) % Вт/(м**2*\celsius)
}

{
	F=(Q*10**3)/(k*dt_ср) % м**2
}

{
	Ks_м=22.4/Re_м**0.25
}

{
	dP_м=Ks_м*(L_п/d_э)*((ro_м*om_м**2)/2)*X_м
}

Второй Вариант {X_м=}

Принимаем {m_м=}

При этом фактическая площадь поперечного сечения пакета со стороны
масла

{
	f_пм=m_м*f_1 % м**2
}

Фактическая скорость движения масла в каналах

{
	om_м=V_м/f_пм % м/с
}

При этих условиях определяем параметры

{
	Re_м=(om_м*d_э)/nu_м
}

{
	Nu_м=0.135*Re_м**0.73*Pr_м**0.43*(Pr_м/Pr_ст)**0.25
}

{
	al_м=(Nu_м*la_м)/d_э % Вт/(м**2*\celsius)
}

{
	k=1/(1/al_к+de_с/la_с+1/al_м) % Вт/(м**2*\celsius)
}

{
	F=(Q*10**3)/(k*dt_ср) % м**2
}

{
	Ks_м=22.4/Re_м**0.25
}

{
	dP_м=Ks_м*(L_п/d_э)*((ro_м*om_м**2)/2)*X_м
}

##Гидромеханический расчет пластинчатого теплообменника

Площадь проходного сечения присоединяемых штуцеров диаметром 150 мм

{
	f_d=(Pi*d_ш**2)/4 % м**2
}

Скорость движения масла в штуцере

{
	om_мш=V_м/f_d % м/с
}

Принимаем {Ks_м=}

{
	dP_ш=Ks_м*(ro_м*om_мш**2)/2 % Па
}

Принимаем {dP_м=}

Общее сопротивление теплообменного аппарата со стороны масла составит

{
	dP=dP_м+2*dP_ш % Па
}

Мощность, необходимая для преодоления потерь давления при прокачивании масла через теплообменник

{
	N_р=(V_м*dP)/Eta_н % Вт
}
